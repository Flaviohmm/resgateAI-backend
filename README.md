## 📦 ResgateAI Backend

Sistema inteligente de **otimização de estoque, energia e sustentabilidade para varejo**, baseado em Spring Boot + IA (Claude) + Motor de Resgate 

---

## 🚀 Visão Geral

O ResgateAI Backend é uma API que conecta três motores principais:

### 🛒 Motor de Resgate
- Detecta produtos próximos ao vencimento
- Calcula risco de perda
- Sugere ações inteligentes:
    - Remarcar preço
    - Criar combos
    - Reposicionar
    - Transformar produtos
    - Doar
### ☀️ Motor Solar (Energia)
- Simula consumo energético da loja
- Integra dados climáticos (Open-Meteo)
- Sugere otimização de energia:
    - Pré-resfriamento
    - Corte de consumo em pico
    - Detecção de anomalias
### 🌱 Sustentabilidade
- Calcula impacto ambiental:
    - Kg de alimentos salvos
    - CO₂ evitado
    - Energia economizada
    - Valor financeiro recuperado
